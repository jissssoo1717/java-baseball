<기능 구현>

1. 상대방(컴퓨터)의 3자리 수 생성
  - 각 세 자리마다 1 ~ 9까지의 난수를 생성한다
  - 난수를 생성 후 중복인지 아닌지 검사한다,
  - 중복이 아니면 수를 추가하여 세 자리 수를 만든다.

2. 사용자 값 입력 받기
  - 세 자리 수를 문자열 형태로 입력받는다.
  - 사용자가 잘못된 값을 입력했을 경우 IllegalArgumentExption 발생
    - 입력받은 문자열의 길이가 3이 아닌 경우
    - 입력된 숫자가 중복일 경우
    - 숫자 외의 값을 입력한 경우
    - 이후 어플리케이션 종료함.
